# Pilates Reservation App

A complete Pilates Studio Booking System.

## Project Structure
- **/frontend**: The main application built with Next.js (Frontend & API).
- **/backend**: (Legacy) The original Go backend. The production version now uses Next.js API Routes for easier serverless deployment.
